🦁 One-time setup!

Vincent's journal uses Firebase (free from Google) to sync across all devices. Follow these steps once — takes about 10 minutes!

1
Go to console.firebase.google.com and click Add project. Name it vincents-journal. Disable Google Analytics. Click Create project.
2
Inside your project, click the Web icon (</>). Give it any nickname and click Register app. You'll see a config object — keep that page open.
3
In the left sidebar go to Build → Firestore Database. Click Create database, choose Start in test mode, pick any location, click Enable.
4
Copy the values from your Firebase config (e.g. apiKey, projectId) into the fields below and click Connect!
API Key
Auth Domain
Project ID
Storage Bucket
Messaging Sender ID
App ID
